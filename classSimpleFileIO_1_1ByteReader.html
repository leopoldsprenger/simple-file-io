Detailed Description

High-performance binary file reader.

Provides efficient sequential access to raw bytes using a manually managed 1 MB buffer to minimize system call overhead.

Note
Intended for large, contiguous binary reads.
Warning
Not safe for concurrent access from multiple threads.
